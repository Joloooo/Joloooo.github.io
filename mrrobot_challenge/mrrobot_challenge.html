Welcome to MR Robot Challenge

The Mr. Robot Challenge is designed to test participants' skills in various aspects of cybersecurity, particularly in brute force attacks, the use of cryptography for hash cracking, and understanding the use of rainbow tables and salting in hashes.

The challenge can be conducted in two paths. The Beginner's path is for those who are new to the above-mentioned concepts, while the Advanced path is for professionals in this field. Both paths ultimately complete the same tasks and achieve the same objectives. The difference is that the Beginner's path provides more information, more hints, and an educational overview of the concepts, whereas the Advanced path offers no hints and is more similar to a traditional CTF challenge.
