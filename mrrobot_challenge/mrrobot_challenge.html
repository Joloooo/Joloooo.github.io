Welcome to MR Robot Challenge

The Mr. Robot Challenge is designed to develop and test participants' skills in various aspects of cybersecurity, particularly in brute force attacks, the use of cryptography for hash cracking, hash identification, and understanding the use of rainbow tables and salting in hashes.

All of the paths eventually achieve the same objective but each path offers different experiences.

The Beginner's path is for those who are new to the concepts, providing more information, hints, and an educational overview.

The Adventurer's path is recommended if you want to not only complete the challenge but to fully grasp the understanding of the concepts discussed in the challenge.

The Advanced path is more like a CTF for experienced individuals who want to refresh their memory in the aforementioned concepts.
